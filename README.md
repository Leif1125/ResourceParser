# ResourceParser
电影及资源搜索器、解析器。

提供最新电影、电视剧的推荐、搜索、下载链接（较稳定）；
提供相当全的各类资源的搜索、下载链接（稳定性由选择数据源而定）。

数据全靠爬，目前仅提供链接，下载请自行使用下载工具。

app/release文件夹下下载apk体验各版本。

目标：
1.爬数据 -> 2.解析出磁力 -> 3.磁力转BT -> 4.BT转下载 -> 5.边下边播
